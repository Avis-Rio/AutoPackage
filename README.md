# 配分表自动转换工具 (AutoPackage)

## 简介
本工具用于将配分表Excel文件自动转换为指定格式的箱設定表格。支持单文件处理和批量文件夹处理。

## 功能特点
*   **格式支持**：完美支持 `.xls` 和 `.xlsx` 格式的配分表。
*   **智能聚合**：自动将同一店铺的多个配货记录合并，并根据SKU配比模式进行智能分组。
*   **批量处理**：支持选择文件夹进行批量转换，自动识别文件名中的ID并生成规范的输出文件名。
*   **安全机制**：输出文件重名检测，提供覆盖或跳过选项。
*   **详细日志**：实时显示处理进度和详细日志，方便排查问题。

## 使用方法

### 1. 单文件转换
1.  打开程序，选择“单文件转换”模式。
2.  点击“浏览”选择配分表文件（源文件）。
3.  程序会自动设置默认的模板路径和输出路径（可手动修改）。
4.  点击“开始转换”。

### 2. 批量转换
1.  选择“批量转换 (文件夹)”模式。
2.  选择包含配分表的文件夹。
3.  （可选）选择统一的输出文件夹。如果留空，转换后的文件将保存在各配分表的同级目录下。
4.  点击“开始转换”。程序将遍历文件夹中的所有Excel文件进行处理。

## 依赖环境
*   Windows 10/11
*   无需安装Python环境（使用打包后的EXE直接运行）

## 注意事项
*   请确保模板文件 `template.xls` 与程序在同一目录下（或手动指定路径）。
*   配分表格式需符合既定规范，否则可能导致读取失败。
